# Interactive AI App

تطبيق ذكاء اصطناعي تفاعلي يساعد المستخدم في:
- تلخيص محاضرات ومحتوى دراسي وعملي
- تحليل صور وإنشاء برومبت تلقائي
- التفاعل مع فيديوهات أو شاشة المستخدم بشكل لحظي (Real-Time)
- محادثة صوتية ودية باللهجة العربية

التقنيات المستخدمة (مبدئياً):
- OpenAI Agents + Realtime API
- Backend (يتم تحديده: Node.js أو .NET)
- Web Frontend (React أو أي إطار آخر لاحقاً)
